# CharitEase

ChartiEase is a mobile app that aims to simplify the process of donating funds and belongings internationally. The goal is for those to be able to give back to their home communities, regardless of international relations or regulations. 

The app also makes it easy for charities and non-profits to keep track of income, donations, and interactions with the communities they serve.

**Features**

- Map featuring drop locations for clothing items.
- International payment processing via Stripe.
- Automated, personalizied charity suggestion algorithm.
- User database & information stored securely via Supabase. 
- Dashboard made for charities to track donations
- Global event tracker for world crises. 
- Social feed for communities to interact with donors.

**Tech Stack**
The React Native and Expo frameworks makes up the base of the app.
The Stripe API is used to process payments securely.
Supabase is used as a database in order to store user and chartiy infromation.
The Google Maps API is used to keep track of clothing donation drop off points in the U.S
